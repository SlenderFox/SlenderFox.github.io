Stock Trader

  • Console application
  • C++

Stock Trader is a small console application written in C++ for an assessment project at AIE.
The game uses simple text input and output to convey important game information; the goal of the game is to make as much money in one year by buying and selling stock in companies.
Stock Trader was written entirely by me and uses no external libraries.
